University project: MongoDB, Express

# Data Management Lab

## Prerequisites

- **Client-Side**
  - Node.js v16.13.1

- **Server-Side**
  - MongoDB (you can use Docker instead)

- **Other**
  - docker
  - docker-compose

- **Launch**
    - clone the repo
    - npm install
    - npm start

    ***Or***
    - docker-compose build
    - docker-compose up

- **Check this out**
   - Open http://localhost:3000/ to check opened project
